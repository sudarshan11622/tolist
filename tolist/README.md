# Advanced Todo List

A feature-rich, responsive todo list application with priority levels, due dates, filtering, and local storage persistence.

## Features

- ✅ Add, edit, and delete tasks
- ✅ Set priority levels (High/Medium/Low)
- ✅ Add due dates with overdue highlighting
- ✅ Filter tasks (All/Pending/Completed/High Priority)
- ✅ Local storage persistence
- ✅ Responsive design
- ✅ Keyboard shortcuts
- ✅ Task statistics
- ✅ Clean, modern UI
